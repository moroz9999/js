1	var a = 1, b = 1, c, d;
2	c = ++a; alert(c);           // 2
3	d = b++; alert(d);           // 1
4	c = (2+ ++a); alert(c);      // 5
5	d = (2+ b++); alert(d);      // 4
6	alert(a);                    // 3
7	alert(b);                    // 3

Такие "странные" результаты работы кода объясняются пост- и префикс- инкрементами, используемыми в разных строках. Так, в строке 2 используется префиксная форма, и увеличение переменной а происходит до присвоения ее значения переменной с. В то же время, в строке 3 переменной d присваивается старое значение переменной b (1) и только после операции присвоения происходит увеличение переменной b на 1.
